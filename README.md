# Evolution Electric Vehicles Website

This project contains a website about the evolution of electric vehicles. When accessing the site, you will be automatically redirected to the main content page.

## Project Structure
- `/evolution-electric-vehicles/` - Main website files
- `index.html` - Redirect page to the main website
- `start_server.py` - Python script to run a local server

## How to Use
There are two ways to access the site:

### Option 1: Direct file access
Simply open the index.html file in a browser or serve the project from this directory. The site will automatically redirect to the main content page.

### Option 2: Using the local server
Run the Python server script to serve the website:
```bash
python3 start_server.py
```
Then visit `http://localhost:8000` in your browser.
