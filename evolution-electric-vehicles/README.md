# Эволюция электромобилестроения

Этот веб-сайт посвящен истории развития электромобилестроения - от первых экспериментальных моделей до современных суперэлектромобилей. 

## Структура проекта

```
evolution-electric-vehicles/
├── index.html          # Основная HTML страница
├── css/
│   └── style.css       # Стили для веб-сайта
├── js/
│   └── script.js       # JavaScript функциональность
├── images/
│   └── .gitkeep        # Пустой файл для сохранения папки в репозитории
└── README.md           # Этот файл
```

## Описание сайта

Сайт включает в себя следующие разделы:

1. **Введение** - Общая информация о проекте
2. **Ранняя история** - Электромобили с 1830-х по 1920-е годы
3. **Современная эпоха** - Возрождение интереса к электромобилям с 1990-х годов
4. **Суперэлектромобили** - Современные высокопроизводительные электромобили
5. **Будущее** - Технологии и направления развития

Каждый раздел содержит информацию о ключевых моделях, технологиях и направлениях развития электромобилей с иллюстрациями (виртуальными, так как реальные изображения не включены в этот проект).

## Технические особенности

- Адаптивный дизайн для различных устройств
- Плавная навигация по разделам
- Интерактивные элементы (увеличение изображений при клике)
- Хронологическая структура с таймлайн-представлением

## Использование

Для просмотра сайта просто откройте `index.html` в любом веб-браузере.
